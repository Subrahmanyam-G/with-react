Introduction

JavaScript is a scripting language. A scripting language is a lightweight programming language.

  • JavaScript code can be inserted into HTML pages.
  • It can be executed by all modern web browsers.
The <script> Tag

Step 1: Add a script tag inside your HTML.

<script type="text/javascript">
// JavaScript code here
</script>
✅ You can place scripts in both the <head> and <body> sections.
Output in JavaScript

1) document.write()

<script>

</script>

2) innerHTML

<h1 id="special"></h1>
<script>
document.getElementById("special").innerHTML = "Welcome to JS Class from Reactify Hub";
</script>

3) window.alert()

Variables & Data Types

Variables are containers for data values. Declare using var, let, or const.

var a = 2;               // Number
var b = "Reactify Hub";  // String
var c = true;            // Boolean
var d = new Array();     // Array

JavaScript is dynamically typed (type is decided at runtime).

  • Number, String, Boolean, Object, Array, Undefined, Null
var length = 16;                              // Number
var lastName = "Johnson";                     // String
var x = { firstName: "John", lastName: "Doe"}; // Object
var c = true;                                 // Boolean
var d = new Array(1,2,3);                     // Array

Operators

Arithmetic

var a = 10, b = 5;
document.write("a+b = " + (a+b));

Comparison

var x=5, y=10;
document.write(x < y); // true

Logical

(x < 10 && y > 1) // AND
(x == 5 || y == 5)   // OR
!(x == y)            // NOT
